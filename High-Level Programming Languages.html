High-Level Programming Languages

High-level programming languages, whereas easy compared to human languages, are a lot of advanced than the languages the pc truly understands, referred to as machine languages. every totally different sort of hardware has its own distinctive machine language.

Lying between machine languages and high-level languages are languages referred to as assembly languages. Assembly languages are just like machine languages, however they're a lot of easier to program in as a result of they permit a computer programmer to substitute names for numbers. Machine languages include numbers solely

Lying on top of high-level languages are languages referred to as fourth-generation languages (usually abbreviated 4GL). 4GLs are so much far from machine languages and represent the category of pc languages nearest to human languages
